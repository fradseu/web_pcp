NOVA EMISSÃO DE ORDEM DE MANUTENÇÃO E FERRAMENTARIA

NOVO MODO DE EMISSÃO DE ORDEM DE MANUTENÇÃO E FERRAMENTARIA.

Descrição e histórico do projeto

O projeto da Folha de manutenção foi criado em meados de março de 2022, quando surgiu a necessidade na fábrica Marflex de documentar todas as ações corretivas realizadas nas ferramentas, para estar em conformidade com as normas ISO.

Escopo do projeto

O projeto WEB PCP tem como objetivo suprir a necessidade de informatizar a documentação. Atualmente a documentação é feita através de documentos Excel, os quais utilizam técnicas de programação VBA. A criação de um sistema web onde podemos centralizar as informações, proporcionando ganho de tempo, pois não será mais necessário procurar pastas dentro da rede ou até mesmo criar itens que já existem por motivos de não serem encontrados. O WEB PCP em seu início deverá conter as funções de CRIAR ORDEM DE MANUTENÇÃO e FERRAMENTARIA, as quais devem ser impressas ou não, guardando as informações em banco de dados PostgreSQL, assim possibilitando as possíveis auditorias em conformidade com a norma ISO.

Requisitos de alto nível

O novo sistema deve incluir o seguinte:

Capacidade de permitir que os usuários internos acessem o aplicativo sem baixar nenhum software.
Conter uma página home, onde possamos criar links para pastas da rede ou documentos.
Capacidade de criar Ordem de Manutenção e ou Ferramentaria.
Capacidade de IMPRIMIR a Ordem de Manutenção e ou Ferramentaria.
Capacidade de ALTERAR a Ordem de Manutenção e ou Ferramentaria.
Capacidade de VISUALIZAR a Ordem de Manutenção e ou Ferramentaria.
Capacidade de DELETAR a Ordem de Manutenção e ou Ferramentaria.
Capacidade de salvar a Ordem de Manutenção e ou Ferramentaria em banco de dados.
Capacidade de exportar dados para geração de indicadores.
Permitir a utilização simultânea do sistema pelos usuários.

Resultados finais

As partes interessadas são a Produção, o Planejamento e controle de Produção, a Qualidade, a Ferramentaria e a Manutenção.

Exclusões exclusivas do escopo

O projeto PCP WEB, será implementado conforme os módulos forem entregues.

Plano de implementação

A nossa proposta é implementar o sistema PCP - WEB, gradativamente. Após a entrega da aplicação iremos realizar um período de testes onde o objetivo é identificar os bugs ou erros de programação. As atualizações serão implementadas conforme necessidade de adequação. A primeira etapa é fornecer a aplicação de Ordem de Serviço Manutenção/Ferramentaria (OSMF), contendo a capacidade de criar, ler, atualizar, deletar e imprimir. A segunda etapa será a atualização da página de listagem de OSMF, onde o objetivo é aplicar técnicas de paginação e filtros para melhor localização dos itens. A terceira etapa, consiste em criar o sistema de usuários cadastrados, onde o usuário deve se logar para utilizar o sistema PCP – WEB.
